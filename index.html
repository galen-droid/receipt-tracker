Setup Guide

1. Go to console.cloud.google.com → create/select a project

2. Enable 3 APIs: Google Drive API, Google Sheets API, Cloud Vision API

3. Go to Credentials → Create OAuth client ID (Web app)

4. Add your hosting URL to both Authorized JavaScript Origins and Redirect URIs

5. Set up OAuth Consent Screen → add team emails as test users

6. Copy the Client ID into Settings in this app
